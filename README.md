# Industry_oriented_Chatbot
A web-based industrial chatbot built with Python and Flask for Retail &amp; E-commerce support across HR, IT Support, and Finance departments.
RetailCorp Support Chatbot
A department-aware support chatbot built with Python (Flask) and plain HTML/CSS/JS — no frameworks, no external dependencies beyond Flask.
Users select a department (HR, IT Support, or Finance) and ask questions in natural language. The bot matches keywords to a structured knowledge base and returns relevant answers instantly.
Departments covered:

HR — Leave policy, salary & payroll, onboarding, HR policies, training
IT Support — Password reset, VPN, hardware, software, email & collaboration
Finance — Invoices, expense claims, budgets, audit & compliance, payments

Tech stack:
Backend: Python, Flask
Frontend: HTML, CSS, JavaScript (vanilla)
No database required — knowledge base is built into the app
